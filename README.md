# Greek Vases

This repository provides the `greek_vases` file containing metadata and photographs of approximately 100 Ancient Greek vases from the British Museum. The images were annotated in **Label Studio** using bounding boxes bbox.

## What is annotated

The annotations include:
- **Named figures (characters)**
- **Clothing items**
- **Everyday objects**
- **Ornamental motifs**

## Important note about image paths

The dataset file contains **local file paths** to the images.  
To use the dataset correctly, please **replace the image paths** with the corresponding paths on your own device.

## Additional tables (other museums)

A separate folder, **"Greek vases in different museums"**, contains tables with links and— for some museums—metadata for Ancient Greek vases from various collections.

The selection criteria were:
- The decoration is sufficiently well preserved.
- The scene includes people.

Vases depicting only animals or ornaments, as well as items whose decoration or shape is too heavily damaged, were excluded.
